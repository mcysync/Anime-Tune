## 💖 About Anime Tune

**Anime Tune** is a gorgeous, anime-inspired revamp of the original **Archive Tune**. We've taken the robust and reliable foundation of the original project and completely transformed the UI/UX. Bathed in pastel pinks, smooth animations, and a cute aesthetic, this fork is perfectly tailored for anime fans, Vtubers, and anyone who loves a cozy, kawaii vibe while browsing their audio archives.

### ✨ What's New in this Fork?

*   **Sakura UI:** A complete overhaul of the color palette featuring soft pastel pinks, magenta accents, and translucent layers.
*   **Anime Aesthetics:** Cute loading screens, floating cherry blossom effects, and custom anime-style icons.
*   **Custom Theming:** Easily toggle between "Daylight Sakura" and "Midnight Magical Girl" (Dark Mode) themes.
*   **All Original Features:** Retains 100% of the core functionality that made Archive Tune great!
